un poligono esta seleccionado cuando:

  • es el poligono que se esta construyendo
  • es el ultimo poligono con el que se interactuo, por ejemplo: borrar un punto, estirar un vertice, moverlo completamente
  • cuando se termina un poligono se comienza la edicion de uno nuevo y este pasa a ser seleccionado

los atributos especiales se aplican a:

  • el proximo poligono que se cree cuando se termine el poligono actual
  • el poligono seleccionado cuando se aprieta el boton "aplicar a poligono seleccionado"

el cambio en la saturacion solo se aplica a el poligono seleccionado

por lo tanto cuando se termina un poligono y se modifica la saturacion, se modifica la saturacion del nuevo poligono que puede no tener un punto y por lo tanto no ser visible

para borrar o editar las posiciones de los puntos de un poligono, este primero tiene que estar terminado
